F

FIFO

First-In-First-Out. The simplest algorithm that replaces the oldest page in memory. Prone to performance anomalies.

L

LRU

Least Recently Used. Replaces the page that hasn't been used for the longest time. A popular approximation of optimal behavior.

O

Optimal

The theoretical best algorithm. It looks into the future to replace the page that won't be used for the longest duration.

What is Belady's Anomaly?

Common logic suggests that giving a program more memory frames should reduce page faults. However, with FIFO, increasing frames can sometimes increase faults. Use our tool to prove this phenomenon.
